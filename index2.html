History

This is a small history about creating FESKA

About Us

I’m an armenian 15 years old boy, living in Yerevan, a student in Shirakatsy Lyceum. I’ m interested in programming. The fact that I was concerned about english level in Armenia was the reason for making this website. Hope that it would be helpful.

Decisions

At first, I thought of making a drone. For 1 month I was getting ready for that, but then I realized that that kind of project is not very affordable for a regular student like me.

Start

So, After cancelling the previous project, I started planning a project, that will match this points down below:
1. Attractive
2. Global problem solving
3. Affordable

Final Decision

Finally, I decided to make an educational website for armenians to learn english for free. Making this website was matching the 3 points, and also it was a good experience for me.

Packaging Design

I planned, that firstly I has had to make the main page. I did it. After doing that, go from optimizing each level to category. There are 4 main categories:
-Work
-Study
-Communication
-Traveling

Optimization

There are 3 main levels in this course: -Beginner, which includes A1 and A2 levels -Intermediate, includes B1 and B2 levels -Advanced, includes C1 and C2 levels
